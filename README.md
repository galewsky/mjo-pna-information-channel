# MJO-PNA Information Channel

This repository contains the analysis code and lightweight reproducibility products for the manuscript:

**The Atmosphere as a Communication Channel: A Matched-Filter Analysis of MJO-PNA Information Transfer**

The analysis builds a phase-lag "codebook" of 500-hPa geopotential-height response templates for MJO-PNA teleconnections, projects daily Z500 anomalies onto those templates, and evaluates phase-resolved mutual information (MI), binary-gated information gain (IG), and communication-inspired reliability metrics.

## Repository Contents

- `src/`: analysis, validation, and figure-generation scripts.
- `docs/reproducible_pipeline.md`: command sequence used for the manuscript analysis.
- `docs/data_access.md`: external data inputs and large intermediate products that are not stored in GitHub.
- `outputs/manuscript_results/`: lightweight generated CSV, TeX, and Markdown outputs used to build manuscript tables and check numerical claims.
- `outputs/surprisal_olr_validation/`: lightweight ROMI/RMM validation summaries.
- `outputs/rmm_sensitivity/`: lightweight sensitivity-summary products.
- `outputs/figures/`: final manuscript figure PNGs small enough for GitHub.

Large raw and intermediate data products are intentionally excluded from GitHub. These include ERA5/CLARA-A3 source files, Zarr stores, receiver NetCDF files, exported codebook NetCDF files, and null-distribution NPZ files.

## Environment

The scripts were run in a Conda environment similar to `environment.yml`. Some figure generation uses Cartopy and Natural Earth coastlines; if Cartopy needs to download shapefiles, set a writable cache location such as:

```bash
export XDG_DATA_HOME=/path/to/writable/cache
export MPLCONFIGDIR=/path/to/writable/mplconfig
```

## Reproducing the Main Tables and Figures

The end-to-end command sequence is documented in:

```bash
docs/reproducible_pipeline.md
```

The central products for the manuscript are generated by:

```bash
python src/phasebank_information_gain.py ...
python src/evaluate_phasebank_skill.py ...
python src/compute_phasebank_telecom_metrics.py ...
python src/make_phasebank_manuscript_figures.py ...
python src/make_additional_review_figures.py ...
```

The `...` arguments are expanded in `docs/reproducible_pipeline.md`.

## Submission Notes

This repository is a code and lightweight-product archive. For fully bitwise reproduction, the large intermediate products listed in `docs/data_access.md` must be regenerated from the external data sources or supplied separately as a data archive.
